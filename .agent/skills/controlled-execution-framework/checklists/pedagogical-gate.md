# Pedagogical Gate Checklist

> Must be completed by the Agent before transitioning from Theory to Execution.

## Pre-Execution Validation
- [ ] Has the root cause of the vulnerability been explicitly explained?
- [ ] Is the payload deconstructed into understandable segments?
- [ ] Has the user been asked to predict the outcome of the execution?
- [ ] Have we tested a benign "Proof of Concept" (e.g., a crash or sleep) before attempting code execution or data exfiltration?

## Error Handling (If execution fails)
- [ ] Was the error output provided to the user without immediate correction?
- [ ] Was a Socratic question asked to guide the user toward identifying the flaw in the payload?
